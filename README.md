Location quiz

## Environments

1. Java

2. Maven

3. MongoDB

## Installation

**1. Clone project**

```bash
git clone https://github.com/ricknguyenit/location-quiz
```

**2. Start api**

```bash
cd api
mvn spring-boot:run
```

**3. Start client**

```bash
cd client
npm install -g @angular/cli
npm install
ng serve
```
Go to <http://localhost:4200>
